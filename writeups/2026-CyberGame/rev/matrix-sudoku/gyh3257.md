# Matrix sudoku

## 문제 설명

> Can you solve this simple matrix? If your input satisfies the constraints, you will get decrypted flag.

`get_flag.py` 제공. 25개의 정수를 쉼표로 입력받아 5×5 그리드를 구성하고, 7개의 제약 조건을 모두 통과하면 AES-CBC로 암호화된 플래그를 복호화해 출력한다.

## 풀이

### 분석

제약 조건:

```python
constraints = [
    ([(0,0)..(0,4)], 15),   # row 0 합
    ([(1,0)..(1,4)], 40),   # row 1 합
    ([(2,0)..(2,4)], 65),   # row 2 합
    ([(3,0)..(3,4)], 90),   # row 3 합
    ([(4,0)..(4,4)], 115),  # row 4 합
    ([(0,0),(1,1),(2,2),(3,3),(4,4)], 65),  # 대각선 합
    ([(0,0),(4,0)], 22),                    # 두 셀 합
]
```

키 생성 방식:

```python
key = SHA256(str(grid))   # Python list repr 그대로 해싱
```

### 풀이 과정

**1단계 — 행 집합 결정**

값 1-25를 각 셀에 한 번씩 배치하는 순열이라고 가정(sudoku).
행 합 조건과 중복 없는 값 제약을 결합하면:

- row 0 합 = 15 → 최솟값 1+2+3+4+5 = 15 → 반드시 `{1,2,3,4,5}`
- row 1 합 = 40 → 나머지 중 최솟값 6+7+8+9+10 = 40 → 반드시 `{6,7,8,9,10}`
- row 3 합 = 90 → 16+17+18+19+20 = 90 → 반드시 `{16,17,18,19,20}`
- row 4 합 = 115 → 21+22+23+24+25 = 115 → 반드시 `{21,22,23,24,25}`
- row 2 합 = 65 → 나머지 `{11,12,13,14,15}`

**2단계 — 고정 셀 결정**

`grid[0][0] + grid[4][0] = 22`에서:
- `grid[0][0] ∈ {1..5}`, `grid[4][0] ∈ {21..25}`
- 합 22를 만족하는 유일한 조합: `grid[0][0]=1`, `grid[4][0]=21`

**3단계 — 대각선 제약**

`1 + grid[1][1] + grid[2][2] + grid[3][3] + grid[4][4] = 65`

각 원소의 범위가 고정되어 있으므로 유효한 대각선 조합은 약 62가지로 줄어든다.

**4단계 — 브루트포스**

각 대각선 조합에 대해 행 내 순열을 열거하고 AES 복호화 시도.
PKCS7 패딩 오류 없이 UTF-8 디코딩 성공 시 플래그 반환.

## 배운 점

- 행 합 조건 + 값 범위 제약 조합으로 탐색 공간을 수학적으로 대폭 축소 가능
- `gen_key`가 `str(grid)` Python repr을 그대로 해싱하므로 리스트 구조와 순서가 키에 직접 영향
- 유효한 AES 패딩 여부를 oracle로 활용하는 브루트포스 기법
